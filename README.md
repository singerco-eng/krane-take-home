# Krane Take-Home

Product direction and working prototype for a six-week build cycle. The procurement log becomes the place PEs work — not just where they start.

**Deliverables:** one-pager (`draft-one-pager.md`), prototype (React + Vite), process note (`process-note.md`)

## Dev Setup

```bash
npm install
npm run dev
```

Runs at `http://localhost:5173`.

### Build

```bash
npm run build
npm run preview
```

### Environment

Copy `.env.local.example` to `.env.local` and fill in the Supabase keys (optional — prototype runs without them).
